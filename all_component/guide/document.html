SCSS 기본사용 규칙
  • 속성값은 반드시 변수로 적용되도록 작성한다. (_variables.scss 에 선언된 변수값을 참고)

    WSG에 따라 필요시 _variables.scss 에 선언된 변수값을 수정/추가 할 수 있다.

  • 엘리먼트 상위에 선언되는 wrap가 있는 클래스는 BEM 방법론 사용한다. 
                                                        .header {
                                                        &__wrap {
                                                            display:flex;
                                                        }
                                                        &__inner {
                                                            display:flex;
                                                        }
                                                    }

                                                    .card {
                                                        &__list {

                                                        }
                                                        &__item {

                                                        }
                                                    }
  • scss 속성 선언시 모든 속성은 소문자로 작성되어야 하며, 선언된 속성과 속성 사이는 빈칸으로 구분하고 속성과 속성값 사이에 빈칸은 적용하지 않는다. 
                                                        /* 잘못된 사용 예
                                                    1. 속성과 속성값 사이 빈칸 적용
                                                    2. 속성에 대문자 사용
                                                    */
                                                    .example {display: block; Background: $color-white; COLOR: $color-black;}
                                                    
                                                    /* 올바른 사용 예 */
                                                    .example { display:block; background:$color-white; color:$color-black; }
  • 페이지내 style 사용은 금하며, 예외적으로 input 또는 팝업(레이어팝업 포함) 사이즈 지정이 필요한 경우 style="width" 사용할 수 있다.
  • 문서내 margin/padding 사용이 필요할 경우 해당 요소의 클래스내에서 처리하며, 아래 예시와 같이 inner style 또는 공통클래스(ex. marT15, padL20 등)를 만들어 사용하지 않는다 
                                                        <!-- 잘못된 사용 예 -->
                                                    <p style="margin-top:20px; padding-bottom:15px;">사용금지</p>
                                                    <p class="marT20 padL20">사용금지</p>
                                                    
                                                    <!-- 올바른 사용 예 -->
                                                    p.sampleBox {margin-top:20px; padding-bottom:15px;}
                                                    <p class="sampleBox">inner style 또는 공통클래스(ex. marT15, padL20 등)를 만들어 사용하지 않는다.</p>
